# DataFootprint Auditor
Aplicación de escritorio diseñada para auditar tu privacidad digital. Permite importar, clasificar y gestionar datos recopilados por empresas tecnológicas para visualizar tu nivel de exposición.

## Tecnologías 
* **Lenguage:** Java (JavaFX)
* **Base de datos:** MySQL / MariaDB

## Funciones Principales
* **Importación:** Carga masiva de datos de rastreo mediante archivos CSV.
* **Gestión Local CRUD:** Visualiza, añade, edita el nivel de sensibilidad y elimina registros de forma segura en tu propio equipo.
* **Análisi visual:** Filtros por empresa y alertas en rojo para datos de "Alta Sensibilidad".
* **Exportación:** Generación de informes de privacidad en formato `.txt`.

## Cómo probar el proyecto
1. Clona este repositorio en tu equipo.
2. Configura la conexión a tu base de datos local MySQL/MariaDB.
3. Ejecuta la aplicación desde tu entorno de desarrollo favorito (ej. Eclipse, IntelliJ, NetBeans, Visual Studio Code).
